## üéØ SIMPLE STEPS

### **Step 1: Open Each HTML File**
1. Go to your `stage2/prototypes/` folder
2. Double-click each HTML file to open in your web browser
3. You'll see a mockup of your dashboard/interface

### **Step 2: Take Screenshots**
1. Make sure the browser window is full-screen
2. Press `Print Screen` (or `Cmd+Shift+4` on Mac)
3. Save the image as PNG

### **Step 3: Save Images**
1. Create a new folder: `stage2/prototype_images/`
2. Save each screenshot with these names:
   - `prototype_01_dashboard.png`
   - `prototype_02_realtime.png`
   - `prototype_03_component.png`
   - `prototype_04_anomaly.png`
   - `prototype_05_import.png`

- The HTML files are self-contained
- They should show mockups of your dashboard
- If they look broken, check the file path
